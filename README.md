# SalesDashboard
Microsoft Excel Sales Dashboard

In this project, I created a Microsoft Excel sales dashboard featuring pivot charts and slicers. I also looked at Excel's three-dimensional animated map. Here is a look at the end result:

<img src=https://github.com/WilliamJMora/SalesDashboard/blob/main/Pictures/Dashboard1.png>

## Table of Contents ##

1. Data Cleaning
2. Pivot Tables
3. Pivot Charts
4. Dashboard Creation
5. 3D Mapping

First, I will go over how I prepared and analyzed the data with power query. Next, I will show how pivot tables were used to pull relevant data from the worksheet and how they are subsequently used to create pivot charts. Then, the pivot charts will be applied onto the dashboard. Lastly, I wanted to see how Excel's 3D mapping could be used to visualize the data through animation.
